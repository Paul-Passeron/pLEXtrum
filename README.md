# pLEXtrum
Header only rule / action based lexing library for C.

## Usage
Simply add ```plextrum.h``` in your project and include it.

### As a header
```C
#include "plextrum.h"
```

### As implementation
```C
#define PLEXTRUM_IMPL
#include "plextrum.h"
```
